# Snowflake End-to-End Data Pipeline

## Project Overview
This project demonstrates building an end-to-end data pipeline using Snowflake, AWS S3, Snowpipe, Streams, Tasks and DBT.

## Architecture
Source → AWS S3 → Snowpipe → Raw Layer → Streams → Tasks → DBT → Mart Layer

## Technologies Used
- Snowflake
- AWS S3
- Snowpipe
- Streams & Tasks
- DBT
- SQL
